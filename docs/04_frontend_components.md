# Frontend Components

## FilmCard
- Poster image
- Title + Team Name
- Members + roles
- Video iframe

## FilmGrid
- Responsive grid (desktop 3 cols, mobile 1 col)
- Maps through FilmCard components

## RegistrationForm
- Team Name
- Email
- Password
- Members list with roles

## SubmissionForm
- Auto-filled teamName and members[] from JWT
- Film title
- Poster GDrive link
- Video GDrive link
